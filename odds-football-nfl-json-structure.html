Available Game Statuses

[scheduled] | scheduled to occur

[created] | game created, begun loggin info.

[inprogress] | first pitch of game has been thrown

[halftime] | game currently at halftime

[cancelled] | game cancelled, no makeup game

[delayed] | start of game delayed or switched from inprogress to dealyed for some reason

[postponed] | game postponed and will be made up

[time-tbd] | game scheduled, but time has not been announced

[flex-schedule] Scheduled, but will be moved most likely for broadcast purposes
