Weather Maps Images v1

Query Structure
Update Frequency:
15 Minutes
API:
https://api.digichief.com/weathermaps/getweathermap/{customer}/{customer_api_key}?latlon={XX.XXXX,-XX.XXXX]&radius={In Miles}&size={PixelxPixel}&type={numeric value}&city={optional parameters}
Parameters
Query Parameter Description | Parameter Options

customer

Unique customer name provided by Digichief

customer_api_key

Your company's unique API key

latlon

Coordinates for the central point of the requested map. XX.XXXX,-XX.XXXX

radius

Radius, in miles from the Lat/Lon

size

Resolution width and height of the image, such as 1280x720

type

Use numeric value. Map types include 0-Radar, 1-Watch/Warn, 2-Satellite, 3-Hi/Low, 4, Highs, 5-Lows, 6-Rain, 7-Wind, 8-Current Conditions, 9- Lightning Strikes, 10-Today’s Forecast, 11-Tomorrow’s Forecast

city

City,State or Comma-delimited string of pairs

Optional Parameter

counties

Enables a county border layer

format

JPG or GIF, Defaults to JPG

metric

Switches units to metric where applicable
