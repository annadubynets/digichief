parameters

Mandatory Parameter

airport=

IATA Airport Codes

Bring this up in a separate tab page

https://en.wikipedia.org/wiki/List_of_airports_by_IATA_airport_code:_A

IATA airport codes (you can enter up to 10 separated by commas)

Note: You can enter up to 10 IATA airport codes separated by commas

Optional Parameters

format=

XML or JSON (returns JSON if not used)

arr_dep=

Arrival, departure or both. If not used, it defaults to both

airline=

IATA Airline Codes

Bring this up in a separate tab page

https://en.wikipedia.org/wiki/List_of_airline_codes#Codes

IATA Airline codes if you want information for a particular airline.

Note: You can enter multiple IATA codes separated by commas or ALL
