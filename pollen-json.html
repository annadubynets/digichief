Pollen v1

Query Structure
Update Frequency:
5 minutes
API:
https://api.digichief.com/pollen/getpollen/{customer}/{customer_api_key}?{parameter}&{optional_parameters}
Parameters
Query Parameter Description | Parameter Options

customer

Unique customer name provided by Digichief

customer_api_key

Your company's unique API key | Digichief will give this to you

parameters

Mandatory Parameter

?location=40509

Zip Code or City,State

Optional Paramter

&format=json

XML or JSON - Default is JSON
Data Points Returned
Data Points Description

content

datafeed

pubDate

Published Date, Time in GMT

latitude

Lattitude

longitude

Longitude

distribution

Distribution By

source

Attribution

pollen_forecast

Forecast Date

forecastdate

Forecast Date

city

Returns Zip Code from Request in API

pollen_day0

Pollen for Today

value

Pollen Index Value

text

Pollen Risk: Low, Low-Medium, Medium, Medium-High, High

predom_pollen

Predominant Pollen type present

description

0-2.4 (Low) - Pollen levels between 0 and 2.4 tend to affect very few individuals among the allergy-suffering public.

2.5-4.8 (Low-Medium) - Pollen levels between 2.5 and 4.8 tend to start affecting individuals extremely sensitive to the predominant pollen.

4.9-7.2 (Medium) - Pollen levels between 4.9 and 7.2 will likely cause symptoms for many individuals who suffer from allergies to the predominant pollen types of the season.

7.3-9.6 (Medium-High) - Pollen levels between 7.3 and 9.6 tend to affect a large number of individuals who suffer from the pollen types of the season.

9.7-12.0 (High) - Pollen levels between 9.7 and 12.0 tend to affect most individuals who suffer from the pollen types of the season. Symptoms may become more severe during days with high pollen levels

Provides an assesment of the current pollen forecast. Please see the following table for the various desciptions
