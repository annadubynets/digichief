Optional attribute that converts to your desired currency. For example &currency=CAD would convert to Canadian Currency. Please note that stock pricing by default is supplied as the native exchange currency, for example NYSE is supplied as USD. If the parameter is set with "two" dash-separated three-letter currency codes, say CAD-JPY, it converts all price-related values that are in the first specified currency to the second specified currency.
